## 1. Natural Language Processing & Large Language Models

### 1.1 Foundational Papers

**Attention Is All You Need**
- **Authors**: Vaswani, A., Shazeer, N., Parmar, N., et al.
- **Year**: 2017
- **Publication**: NeurIPS 2017
- **arXiv**: [1706.03762](https://arxiv.org/abs/1706.03762)
- **Key Contributions**: 
  - Introduced the Transformer architecture
  - Self-attention mechanism
  - Multi-head attention
  - Positional encoding
- **Relevance to DeepDive AI**: Foundation for Gemini and all modern LLMs used in the platform

**BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding**
- **Authors**: Devlin, J., Chang, M.W., Lee, K., Toutanova, K.
- **Year**: 2018
- **Publication**: NAACL 2019
- **arXiv**: [1810.04805](https://arxiv.org/abs/1810.04805)
- **Key Contributions**:
  - Bidirectional context understanding
  - Masked language modeling
  - Next sentence prediction
- **Relevance**: Text analysis, sentiment analysis, contextual understanding

**Language Models are Few-Shot Learners (GPT-3)**
- **Authors**: Brown, T., Mann, B., Ryder, N., et al.
- **Year**: 2020
- **Publication**: NeurIPS 2020
- **arXiv**: [2005.14165](https://arxiv.org/abs/2005.14165)
- **Key Contributions**:
  - 175B parameter model
  - In-context learning
  - Few-shot prompting
- **Relevance**: Conversational AI, content generation, prompt engineering

### 1.2 Advanced Language Models

**PaLM: Scaling Language Modeling with Pathways**
- **Authors**: Chowdhery, A., Narang, S., Devlin, J., et al.
- **Year**: 2022
- **arXiv**: [2204.02311](https://arxiv.org/abs/2204.02311)
- **Key Contributions**: 540B parameter model, breakthrough performance
- **Relevance**: Understanding Google's AI capabilities

**LLaMA: Open and Efficient Foundation Language Models**
- **Authors**: Touvron, H., Lavril, T., Izacard, G., et al.
- **Year**: 2023
- **arXiv**: [2302.13971](https://arxiv.org/abs/2302.13971)
- **Key Contributions**: Efficient open-source models
- **Relevance**: Alternative model options, Groq API support

**Gemini: A Family of Highly Capable Multimodal Models**
- **Authors**: Google DeepMind Team
- **Year**: 2023
- **Link**: [Google AI Blog](https://blog.google/technology/ai/google-gemini-ai/)
- **Key Contributions**: Multimodal understanding, advanced reasoning
- **Relevance**: Primary AI model used in DeepDive AI

## 2. Information Retrieval & Search

### 2.1 Classical Information Retrieval

**The PageRank Citation Ranking: Bringing Order to the Web**
- **Authors**: Page, L., Brin, S., Motwani, R., Winograd, T.
- **Year**: 1999
- **Publication**: Stanford InfoLab
- **Link**: [Stanford Technical Report](http://ilpubs.stanford.edu:8090/422/)
- **Key Contributions**:
  - Web page ranking algorithm
  - Link analysis
  - Random surfer model
- **Relevance**: Understanding search engine result ranking

**The Probabilistic Relevance Framework: BM25 and Beyond**
- **Authors**: Robertson, S., Zaragoza, H.
- **Year**: 2009
- **Publication**: Foundations and Trends in Information Retrieval
- **Key Contributions**:
  - BM25 ranking function
  - Term frequency-inverse document frequency
- **Relevance**: Document ranking, search result scoring

### 2.2 Neural Information Retrieval

**Dense Passage Retrieval for Open-Domain Question Answering**
- **Authors**: Karpukhin, V., Oguz, B., Min, S., et al.
- **Year**: 2020
- **Publication**: EMNLP 2020
- **arXiv**: [2004.04906](https://arxiv.org/abs/2004.04906)
- **Key Contributions**:
  - Dense vector representations
  - Bi-encoder architecture
  - Improved retrieval accuracy
- **Relevance**: Semantic search, question answering

**ColBERT: Efficient and Effective Passage Search via Contextualized Late Interaction**
- **Authors**: Khattab, O., Zaharia, M.
- **Year**: 2020
- **Publication**: SIGIR 2020
- **arXiv**: [2004.12832](https://arxiv.org/abs/2004.12832)
- **Key Contributions**: Late interaction mechanism, efficient retrieval
- **Relevance**: Fast semantic search

**REALM: Retrieval-Augmented Language Model Pre-Training**
- **Authors**: Guu, K., Lee, K., Tung, Z., et al.
- **Year**: 2020
- **Publication**: ICML 2020
- **arXiv**: [2002.08909](https://arxiv.org/abs/2002.08909)
- **Key Contributions**: Retrieval-augmented generation
- **Relevance**: Combining search with language models

## How to Use This Bibliography

1. **For Understanding Core Concepts**: Start with foundational papers (Transformer, BERT, GPT-3)
2. **For Implementation Details**: Refer to specific technique papers (RAG, Dense Retrieval)
3. **For Best Practices**: Review survey papers and industry blogs
4. **For Staying Current**: Follow conference proceedings and AI blogs

## Citation Format

When citing these papers in your own work, use standard academic citation formats (APA, IEEE, etc.). Most papers include BibTeX citations on their arXiv pages.
